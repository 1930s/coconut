Coconut

Coconut is a variant of Python built for simple, elegant, Pythonic functional programming.

Coconut is hosted on PyPI, where it has been downloaded over 20,000 times. Installing Coconut is as easy as:

python -m pip install coconut

If you’re new to Coconut, a straightforward, easy-to-follow tutorial of the Coconut Programming Language is provided in the Coconut Help.

If you’re looking for info about a specific feature, Coconut’s complete documentation can be accessed through the Coconut Docs.

If you’re having a problem with Coconut, please create a new issue detailing the problem, and it will be addressed as soon as possible.

Coconut is completely open-source, and new contributors are always welcome. Contributing to Coconut is as simple as forking Coconut on GitHub, making your changes, and proposing a pull request.
